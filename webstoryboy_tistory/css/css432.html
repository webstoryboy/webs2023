column-rule-style

column-rule-style 속성은 컬럼 라인의 스타일을 정의합니다.

특징 설명
기본 값 column-rule-style : none;
적용 column
버전 CSS3

정의(Definition)

  • column-rule-style 속성은 컬럼 라인의 스타일을 정의합니다.

문법(Syntax)

column-rule-style : none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset

/* 테두리 스타일 속성을 사용하는 경우 */
column-rule-style: none;
column-rule-style: hidden;
column-rule-style: dotted;
column-rule-style: dashed;
column-rule-style: solid;
column-rule-style: double;
column-rule-style: groove;
column-rule-style: ridge;
column-rule-style: inset;
column-rule-style: outset;

/* 글로벌 속성을 사용하는 경우 */
column-rule-style: inherit;
column-rule-style: initial;
column-rule-style: unset;

속성(Property)

속성값 설명
none 컬럼 테두리 스타일을 설정하지 않습니다.
hidden 컬럼 테두리 스타일을 숨김으로 설정합니다.
dotted 컬럼 테두리 스타일을 dotted(점선)로 설정합니다.
dashed 컬럼 테두리 스타일을 dashed(긴 점선)로 설정합니다.
solid 컬럼 테두리 스타일을 solid(실선)로 설정합니다.
double 컬럼 테두리 스타일을 double(이중선)로 설정합니다.
groove 컬럼 테두리 스타일을 groove(움푹 들어간 입체적인 선)로 설정합니다.
ridge 컬럼 테두리 스타일을 ridge(앞으로 튀어나온 입체적인 선)로 설정합니다.
inset 컬럼 테두리 스타일을 inset(박스 전체가 움푹 들어간 입체적인 선)로 설정합니다.
outset 컬럼 테두리 스타일을 outset(박스 전체가 앞으로 튀어나온 입체적인 선)로 설정합니다.
inherit 컬럼 테두리 스타일의 속성 값을 상위요소한테 상속받습니다.
