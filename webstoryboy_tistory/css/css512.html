outline-style

outline-style 속성은 아웃라인 스타일을 정의합니다.

특징 설명
기본 값 outline-style : none
버전 CSS3

정의(Definition)

  • outline-style 속성은 아웃라인 스타일을 정의합니다.

문법(Syntax)

outline-style : none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset

속성(Property)

속성값 설명
none outline 스타일을 설정하지 않습니다.
hidden outline 스타일을 숨김으로 설정합니다.
dotted outline 스타일을 dotted(점선)로 설정합니다.
dashed outline 스타일을 dashed(긴 점선)로 설정합니다.
solid outline 스타일을 solid(실선)로 설정합니다.
double outline 스타일을 double(이중선)로 설정합니다.
groove outline 스타일을 groove(움푹 들어간 입체적인 선)로 설정합니다.
ridge outline 스타일을 ridge(앞으로 튀어나온 입체적인 선)로 설정합니다.
inset outline 스타일을 inset(박스 전체가 움푹 들어간 입체적인 선)로 설정합니다.
outset outline 스타일을 outset(박스 전체가 앞으로 튀어나온 입체적인 선)로 설정합니다.
