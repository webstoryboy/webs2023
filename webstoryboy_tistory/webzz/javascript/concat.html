concat()

concat() 메서드는 둘 이상의 문자열을 결합합니다.

특징 설명
유형(type) String
버전(version) ES1
결과값(return) 문자열(string)
사용성 ★★★☆☆

문법(Syntax)

"문자열".concat(문자열, 문자열,....) 

const text1 = "javascript";
const text2 = "reference";

text1.concat(text2) //javascriptreference
text1.concat(' ', text2) //javascript reference
text1.concat(', ', text2) //javascript, reference
text1.concat('+', text2) //javascript+reference
  • 문자열(string) : 문자열을 설정합니다.

정의(Definition)

  • concat() 메서드는 둘 이상의 문자열을 결합합니다.
  • concat() 메서드는 기존의 문자열을 변경하지 않습니다.
  • concat() 메서드는 새로운 문자열을 반환합니다.
