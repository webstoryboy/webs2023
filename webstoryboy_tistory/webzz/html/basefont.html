<basefont>

<basefont> 태그는 문서의 기본 폰트, 사이즈, 종류를 설정합니다.

특징 설명
요소 인라인 요소(Inline Element)
닫는 태그 Self-closing(<base />)
버전 HTML4(HTML5에서 지원하지 않습니다.)
시각적 표현 -
사용성 -

정의(Definition)

  • <basefont> 태그는 문서의 기본 폰트, 사이즈, 종류를 설정합니다.
  • <basefont> 태그는 <head> 섹션 안에 설정합니다.
  • <basefont>는 HTML5에서 지원되지 않으며, 그 기능은 CSS가 대신 합니다.(font, font-family, font-size, font-size-adjust, font-style, font-variant, font-weight, @font-face)

속성(Property)

속성 값(예) 설명 버전
color color="red" 폰트 컬러를 설정합니다.
face face="serif" 폰트 종류를 설정합니다.
size size="8" 폰트 사이즈를 설정합니다.
