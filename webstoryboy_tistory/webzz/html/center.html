<center>

<center> 태그는 텍스트를 가운데 정렬합니다. CSS 속성이 기능을 대신하고 있기 때문에 현재는 사용하지 않습니다.

<center>

<center> 태그는 텍스트를 가운데 정렬합니다.

특징 설명
요소 블록 요소(Block Element)
닫는 태그 적용(<center> ~ </center>)
버전 HTML4(HTML5에서 지원하지 않습니다.)
시각적 표현 가운데 정렬
사용성 ☆☆☆☆☆

정의(Definition)

  • <center> 태그는 가운데 정렬을 할 때 사용합니다.
  • <center> 태그는 HTML5에서 지원하지 않습니다. 대신 CSS를 사용합니다.
  • 인라인 구조에서 가운데 정렬은 CSS의 text-align : center를 사용합니다.
  • 블록 구조에서 가운데 정렬은 CSS의 margin: 0 auto를 사용합니다.
  • display: flex와 dispaly: grid를 통해서도 가운데 정렬이 가능합니다.
