.charAt() 메서드는 문자열에서 인덱스 번호에 해당하는 문자를 반환합니다.

특징 설명
반환값(returns) 유니코드 단일문자

문법(Syntax)

String.charAt(인덱스)

  • 인덱스(index) : 지정된 인덱스 값을 설정합니다.

정의(Definition)

  • .charAt() 메서드는 문자열에서 인덱스 번호에 해당하는 문자를 반환합니다.
