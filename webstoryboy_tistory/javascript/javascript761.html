.prompt()

.prompt() 메서드는 텍스트를 입력 할 수 있는 대화 상자를 표시합니다.

특징 설명
반환값(returns) 사용자가 입력한 문자열 또는 null

문법(Syntax)

window.prompt("내용","기본값")

  • 내용(message) : 사용자에게 보여줄 문자열을 설정합니다.
  • 기본값(default) : 텍스트 입력 필드에 기본으로 채워 넣을 문자열을 설정합니다.

정의(Definition)

  • .prompt() 메서드는 텍스트를 입력 할 수 있는 대화 상자를 표시합니다.
