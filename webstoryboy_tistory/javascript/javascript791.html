.endsWith()

.endsWith() 메서드는 문자열에서 특정 문자열로 끝나는지를 확인합니다.

특징 설명
반환값(returns) true or false

문법(Syntax)

String.endsWith(검색 문자열, 길이 값)

  • 검색 문자열(searchString) : 검색하고 싶은 문자열을 설정합니다.
  • 길이 값(length) : 길이 값을 설정합니다.

정의(Definition)

  • .endsWith() 메서드는 문자열에서 특정 문자열로 끝나는지를 확인합니다.
