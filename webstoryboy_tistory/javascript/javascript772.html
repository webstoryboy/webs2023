[JavaScript] .lastIndexOf()

.lastIndexOf() 메서드는 지정된 요소가 배열에서 발견 될 수있는 마지막 인덱스를 반환하고, 존재하지 않으면 -1을 반환합니다.

.lastIndexOf()

.lastIndexOf() 메서드는 지정된 요소가 배열에서 발견 될 수있는 마지막 인덱스를 반환하고, 존재하지 않으면 -1을 반환합니다.

특징 설명
반환값(returns) 배열에서 찾은 요소

문법(Syntax)

Array.lastIndexOf(검색 요소, 시작 요소)

  • 검색 요소(searchElement) : 검색할 요소를 설정합니다.
  • 시작 요소(fromIndex) : 검색을 시작할 위치를 설정합니다.

정의(Definition)

  • .lastIndexOf() 메서드는 지정된 요소가 배열에서 발견 될 수있는 마지막 인덱스를 반환하고, 존재하지 않으면 -1을 반환합니다.
