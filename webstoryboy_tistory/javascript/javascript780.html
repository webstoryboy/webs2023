[JavaScript] .slice()

.slice() 메서드는 배열의 일부분을 선택하여 새로운 배열 객체로 반환합니다.

.slice()

.slice() 메서드는 배열에서 첫 번째 요소를 제거하고, 제거된 요소를 반환합니다.

특징 설명
반환값(returns) 새로운 배열

문법(Syntax)

Array.slice(시작, 끝)

  • 시작(begin) : 시작 인덱스 값을 설정합니다.
  • 끝(end) : 끝 인덱스 값을 제외하고 추출 할 인덱스를 설정합니다.

정의(Definition)

  • .slice() 메서드는 배열의 일부분을 선택하여 새로운 배열 객체로 반환합니다.
