.map()

.map() 메서드는 배열 내의 모든 요소 각각에 대하여 주어진 함수를 호출한 결과를 모아 새로운 배열을 반환합니다.

특징 설명
반환값(returns) 새로운 배열

문법(Syntax)

Array.map(콜백함수(현재값, 인덱스, 배열))

  • 콜백함수(callback) : 함수가 실행됩니다.
  • 현재값(currentValue) : 현재 처리 될 값을 반환합니다.
  • 인덱스(index) : 인덱스 값을 반환합니다.
  • 배열(array) : 배열 값을 반환합니다.

정의(Definition)

  • .map() 메서드는 배열 내의 모든 요소 각각에 대하여 주어진 함수를 호출한 결과를 모아 새로운 배열을 반환합니다.
