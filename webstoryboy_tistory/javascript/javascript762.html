.fill()

.fill() 메서드는 배열의 시작 인덱스부터 끝 인덱스까지 하나의 값으로 채웁니다.

특징 설명
반환값(returns) 새로운 배열 객체

문법(Syntax)

Array.fill(속성[, 시작값[, 끝값]])

  • 속성(value) : 배열에 채울 값을 설정합니다.
  • 시작값(start) : 시작 인덱스 값을 설정합니다.
  • 끝값(end) : 끝 인덱스 값을 설정합니다.

정의(Definition)

  • .fill() 메서드는 배열의 시작 인덱스부터 끝 인덱스까지 하나의 값으로 채웁니다.
