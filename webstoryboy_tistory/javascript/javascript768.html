.includes()

.includes() 메서드는 배열에 특정 요소를 포함하고 있는지 판별합니다.

특징 설명
반환값(returns) 불린(Boolean)

문법(Syntax)

Array.includes(찾는 값, 시작 값)

  • 찾는 값(valueToFind) : 탐색할 요소
  • 시작 값(fromIndex) : 검색을 시작할 위치를 설정합니다.

정의(Definition)

  • .includes() 메서드는 배열에 특정 요소를 포함하고 있는지 판별합니다.
