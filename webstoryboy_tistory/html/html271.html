<figure> 태그는 인용, 도표, 사진, 코드 등의 독립적인 콘텐츠를 설정합니다.

특징 설명
요소 블록 요소(Block Element)
닫는 태그 닫는 태그 사용(<figure> ~ </figure>)
버전 HTML5
사용성

정의(Definition)

  • <figure> 태그는 인용, 도표, 사진, 코드 등의 독립적인 콘텐츠를 설정합니다.
  • <figure> 태그는 <figcaption> 요소를 사용해 설명을 포함할 수 있습니다.
