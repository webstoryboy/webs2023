<blockquote>

<blockquote> 태그는 긴 문장의 인용문을 설정합니다.

특징 설명
요소 블록 요소(block Element)
닫는 태그 닫는 태그 사용(<blockquote> ~ </blockquote>)
버전 HTML4
시각적 표현 여백이 생김
사용성

정의(Definition)

  • 인용(quotation)을 표현하는 태그는 <blockquote> 태그와 <q> 태그가 있습니다.
  • <blockquote> 태그는 전체 문장에 해당하는 긴 인용문에 사용합니다.
  • <blockquote> 태그는 인용된 정보 출처는 cite 속성을 통해 설정합니다.
  • <blockquote> 태그를 사용하면 자동으로 여백이 생깁니다. 여백이 목적이라면 CSS를 사용합니다.
  • <blockquote> 태그는 블록 요소이며, <q> 태그는 인라인 요소입니다.
  • <figure>, <figcaption> 태그를 같이 사용하여, 인용문을 더 정확하게 사용할 수 있습니다.

비교(Compare)

  • 짧은 인용문을 설정 한다면 <q> 태그를 사용합니다.
  • 긴 인용문을 설정 한다면 <blockquote> 태그를 사용합니다.
  • 인용한 작품의 출처를 설정 한다면 <cite> 태그를 사용합니다.

속성(Property)

속성 값(예) 설명 버전
cite cite="URL" 인용된 사이트 주소를 설정합니다. -
