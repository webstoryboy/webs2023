<datalist>

<datalist> 태그는 선택 목록과 자동 완성 기능을 설정합니다.

사용자가 데이터를 입력하면 일치하는 목록이 있을 경우 보여줍니다. <select> 태그는 주어진 선택 목록을 선택만 할 수 있지만, <datalist> 태그는 선택 목록에 없어도, 사용자가 입력할 수 있습니다.

정의(Definition)

  • <datalist> 태그는 태그는 선택 목록과 자동 완성 기능을 설정합니다.
  • <datalist> 태그는 <input> 요소의 list 속성 값과 datalist의 id 값을 연결하여 사용합니다.
  • <datalist> 태그는 사용자가 데이터를 입력하면 목록에서 일치하는 데이터를 찾아서 자동완성 기능을 제공합니다.
  • <datalist> 태그는 선택 목록이 많을 경우에 사용자가 데이터를 쉽게 찾아서 선택할 수 있습니다.
  • <datalist> 태그는 선택 목록에 없는 값도, 사용자의 입력 할 수 있습니다.
  • <selecg> 태그와 <datalist> 태그는 선택 목록을 보여주는 것은 똑같지만, <datalist>는 사용자가 선택 목록에 없는 값도 사용할 수 있으면, 일치하는 데이터 값이 있으면 바로 선택할 수 있습니다.

비교(Compare)

  • <select> 태그는 선택 목록을 설정합니다.
  • <datalist> 태그는 선택 목록과 자동 완성 기능을 설정합니다.
  • <details> 태그는 접기/펼치기 목록을 설정합니다.
