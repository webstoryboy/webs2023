<body>

<body> 태그는 사이트 본문에 해당하는 모든 콘텐츠를 포함하는 섹션을 설정합니다.

특징 설명
요소 블록 요소(Block Element)
닫는 태그 닫는 태그 사용(<body> ~ </body>)
버전 HTML4
사용성

정의(Definition)

  • <body> 태그는 사이트 본문에 해당하는 모든 콘텐츠를 포함하는 섹션을 설정합니다.
  • <body> 태그는 문서에서 한 번만 사용됩니다.

섹션 관련 요소(Section Related Elements)

  • <body> 태그는 사이트 본문에 해당하는 모든 콘텐츠를 포함하는 섹션을 설정합니다.
  • <article> 태그는 사이트의 독립적인 컨텐츠 섹션을 설정합니다.
  • <section> 태그는 주제별 그룹의 콘텐츠 섹션을 설정합니다.
  • <nav> 태그는 내부 페이지 및 외부 페이지에 대한 탐색 링크를 제공하는 페이지 섹션을 설정합니다.
  • <aside> 태그는 주요 콘텐츠의 보조적 컨텐츠 섹션을 설정합니다.
  • <h1>~<h6> 태그는 각 섹션의 제목을 설정합니다.
  • <header> 태그는 사이트의 소개 및 메뉴 그룹을 나타내는 섹션을 설정합니다.
  • <footer> 태그는 사이트 관련 정보 및 관련 링크 콘텐츠 섹션을 설정합니다.

속성(Property)

속성 값(예) 설명 버전
alink alink="black" 문서의 링크의 색상을 설정합니다.
background background="URL" 문서의 배경 이미지를 설정합니다.
bgcolor bgcolor="black" 문서의 백그라운드 색을 설정합니다.
bottommargin bottommargin="10px" 문서의 아래쪽 마진을 설정합니다.
leftmargin leftmargin="10px" 문서의 왼쪽 마진을 설정합니다.
link link="black" 문서에 방문하지 않은 링크의 색상을 설정합니다.
rightmargin rightmargin="10px" 문서의 오른쪽 마진을 설정합니다.
text text="black" 문서의 텍스트 색을 설정합니다.
topmargin topmargin="10px" 문서의 위쪽 마진을 설정합니다.
vlink vlink="black" 문서에 방문한 링크의 색상을 설정합니다.

예제1(Sample)

<body> 태그를 사용하는 예제입니다.
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <title>Document</title>
</head>
<body>
    <!-- 본문에 해당하는 모든 콘텐츠 설정 -->
</body>
</html>
