<blink>

<blink> 태그는 텍스트를 깜박이게 설정합니다. 현재는 사용하지 않으며 오래된 태그입니다. 이 기능은 CSS Animation이 대신 할 수 있습니다.

정의(Definition)

  • <blink> 태그는 텍스트를 깜박이게 합니다.
  • <blink> 태그는 현재 사용하지 않으며, 오래된 태그입니다.
  • CSS Animation을 통해 깜박임 효과를 구현 할 수 있습니다.
