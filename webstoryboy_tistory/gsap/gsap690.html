[GSAP] gsap.fromTo()

애니메이션을 줄 때 시작점과 끝점을 지정해주는 메서드입니다. .to() 메서드는 끝나는 애니메이션 지점을 설정하고, .from() 메서드는 시작하는 애니메이션 지점을 설정하고, .fromTo() 메서드는 시작하는 점과 끝나는 점 모두를 설정할 수 있습니다. 이렇게 하면 CSS 속성을 따로 수정하지 않고 스크립트로 애니매이션을 완벽하게 통제할 수 있습니다.

gsap.fromTo()

gsap.fromTo()은 애니메이션을 설정합니다.

특징 설명
반환값(returns) 움직임(Tween)

문법(Syntax)

gsap.fromTo("타겟", {시작 속성: 시작 속성값, ....},{끝나는 속성: 끝나는 속성값, ....});

  • 타겟(targets): 애니메이션을 주고 싶은 타겟(선택자)을 설정하는 부분입니다. #id, .class를 사용할 수 있으며, 자바스크립트의 document.querySelectorAll()라고 생각하면 됩니다.
  • 시작 속성(values) : CSS의 속성과 속성 값을 주는 부분입니다. CSS 애니메이션 속성과 이름이 약간 다를 수 있지만 대부분 동일하게 사용할 수 있습니다. 여기에는 ease, duration, delay등 여러가지 메서드도 같이 사용할 수 없습니다.
  • 끝나는 속성(values) : CSS의 속성과 속성 값을 주는 부분입니다. CSS 애니메이션 속성과 이름이 약간 다를 수 있지만 대부분 동일하게 사용할 수 있습니다. 여기에는 ease, duration, delay등 여러가지 메서드도 같이 사용할 수 있습니다.

정의(Definition)

  • .fromTo() 메서드는 애니메이션의 시작점과 끝나는점을 모두 설정할 수 있습니다.
  • .fromTo() 메서드는 .to() 메서드와 .from() 메서드의 집합체라고 생각하면 됩니다.
  • .fromTo() 메서드는 CSS의 애니메이션 관련된 모든 속성을 사용할 수 있습니다.
  • .fromTo() 메서드를 제어하려면, 여러가지 컨트롤 메서드를 사용할 수 있습니다.
