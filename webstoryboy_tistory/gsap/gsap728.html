.seek()

.seek()는 애니메이션을 특정 시간으로 이동합니다.

특징 설명
반환값(returns) 자신값(Self)

문법(Syntax)

.seek(시간, 이벤트);

  • 시간(time) : 이동할 시간을 설정합니다.
  • 이벤트(suppressEvents) : 기본값(true) : 기본값이면 애니메이션이 끝난 후 이벤트나 콜백이 일어나지 않습니다.

정의(Definition)

  • .seek()는 애니메이션을 특정 시간으로 이동합니다.
