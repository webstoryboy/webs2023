gsap.getProperty()

gsap.getProperty()은 속성 값을 가져옵니다.

특징 설명
반환값(returns) 모든 속성 값

정의(Definition)

  • .getProperty() 메서드는 현재 속성 값을 가져옵니다.
  • .getProperty() 메서드는 속성 값의 특정 단위 값을 가져올 수도 있습니다.
  • .getProperty() 메서드는 단위 값을 생략한다면 number를 가져옵니다. 예를 들어 속성 값을 가져올 때 속성 값이 20px이라면 parseFloat()로 20으로 변경해야 합니다. .getProperty()는 이런 성가로운 부분을 해결해줍니다.
