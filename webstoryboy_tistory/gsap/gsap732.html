[GSAP] .restart()

.restart()는 애니메이션을 처음부터 다시 시작합니다.

.restart()

.restart()는 애니메이션을 처음부터 다시 시작합니다.

특징 설명
반환값(returns) 자신값(Self)

문법(Syntax)

.restart(불린, 불린);

  • 불린(includeDelay) : 기본값(false) : 애니메이션 다시 시작할 때 지연이 필요한 경우 true를 설정합니다.
  • 불린(suppressEvents) : 기본값(true) : 기본값이면 애니메이션이 끝난 후 이벤트나 콜백이 일어나지 않습니다.

정의(Definition)

  • .restart()는 애니메이션을 처음부터 다시 시작합니다.
