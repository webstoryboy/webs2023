[GSAP] gsap.set()

gsap.set()는 속성을 설정합니다.

gsap.set()

gsap.set()은 속성 값을 설정합니다.

특징 설명
반환값(returns) 움직임(Tween)

정의(Definition)

  • .set() 메서드는 속성을 설정합니다. 속성 값을 가져오는 .getProperty() 메서드와 비교됩니다.
  • .set() 메서드는 여러 개의 선택자에게 동시에 줄 수도 있습니다.
  • 여러 개의 선택자에게 동시에 반복적으로 설정할 경우 .quickSetter() 메서드를 사용합니다.

비교(Compare)

  • .set() 메서드는 속성을 설정합니다.
  • .getProperty() 메서드는 속성을 가져옵니다.
  • jQuery에서 attr() 메서드는 속성을 설정하거나 속성 값을 반환합니다.
