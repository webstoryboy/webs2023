.addClass()

선택한 요소의 클래스를 추가합니다.

.addClass()

.addClass() 메서드는 선택한 요소의 클래스를 추가합니다.

특징 설명
리턴 값 클래스 추가
버전 1.x
사용성

문법(Syntax)

$("선택자").addClass("클래스이름");
$("선택자").addClass("함수");

정의(Definition)

  • .addClass() 메서드는 선택한 요소의 클래스를 추가합니다.
  • .addClass() 메서드는 클래스 이름을 두 개 이상도 사용할 수 있습니다.
  • .addClass() 메서드는 콜백 함수를 사용 할 수 있습니다.

비교(Compare)

  • .addClass() 메서드는 선택한 요소의 클래스를 추가합니다.
  • .removeClass() 메서드는 선택한 요소의 클래스를 삭제합니다.
  • .toggleClass() 메서드는 선택한 요소의 클래스를 추기/삭제합니다.
  • .hasClass() 메서드는 선택한 요소의 클래스가 있는지 확인합니다.
