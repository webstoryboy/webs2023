 웹 표준이란?
- 웹의 3요소(웹 접근성, 웹 호환성) 중 하나로 기술의 중립성을 나타냄

- 웹 사이트 작성 시 이용하는 HTML, CSS, JavaScript 등에 대한 규정
- 웹 페이지가 다른 기종이나 플랫폼에서도 구현되도록 제작하는 기법 등을 포함

웹 표준을 지킬 경우 장점

- 웹 접근성을 확보할 수 있다

IR 효과란?
- 이미지 대체 텍스트 제공을 위한 CSS 기법
- background 속성으로 이미지를 표현할 경우 대체 문자 표현을 위해 사용

IR 효과의 기법 3가지
- Phark Method : 의미있는 이미지의 대체 텍스트를 제공하는 경우
- WA IR : 의미있는 이미지의 대체 텍스트로 이미지가 없어도 대체 텍스트를 보여주고자 하는 경우
- Screen Out : 대체 텍스트가 아닌 접근성을 위한 숨김 텍스트를 제공하는 경우

/* css */
.ir_pm {display: block; overflow: hidden; font-size: 0; line-height: 0; text-indent: -9999px;}
.ir_wa {display: block; overflow: hidden; position: relative; z-index: -1; width: 100%; height: 100%;} 
.ir_so {overflow: hidden; position: absolute; width: 0; height: 0; line-height: 0; text-indent: -9999px;}
 이미지 스프라이트(Image Sprite) 효과란?
- 이미지를 표현하는 방법 중 background 속성으로 표현하는 방법에서 이용
- 단 하나의 png 파일에 로고나 아이콘과 같은 이미지들을 모아서 생성
- 필요한 만큼의 영역과 좌표를 지정하여 사용

미디어쿼리(Media Query)란?
- 반응형 사이트의 3가지 원칙(유동 레이아웃, 반응형 이미지) 중 하나
- 화면 크기에 따라 CSS를 제작

/* @media [only/not] [media type] [and/.] (조건문) {실행문} */

/* min을 사용한다면 */
@media (min-width: 380px){실행문}
@media (min-width: 768px){실행문}
@media (min-width: 960px){실행문}
@media (min-width: 1280px){실행문}
/* max를 사용한다면 */
@media (max-width: 1280px){실행문}
@media (max-width: 960px){실행문}
@media (max-width: 768px){실행문}
@media (max-width: 380px){실행문}
