MY REACT PROJECT

  • JMusic Site
      React   Axios   styled-components   react-router-dom

    JMusic 사이트는 Youtube Data Api V3 로부터 비디오와 재생목록을 Axios로 가져와서 사용자가 편하게 볼수있게 UI Component를 리엑트를 이용해서 제작하였습니다. 컴포넌트 스타일링은 styled-components를 사용해서 css-in-js 방식으로 스타일링 하였습니다. react-router-dom을 이용해서 Home, Search, Playlists 페이지가 있습니다. 음악을 재생목록에 추가하거나 혹은 음악을 검색하거나 음악을 재생할 수 있는 뮤직 웹앱 SPA사이트를 제작하였습니다. 호스팅은 Netlify 에 하였습니다.

  • NomFlix Site
      React   Axios   styled-components   react-router-dom

    Nomflix 사이트는 The Movie Database Api 로부터 영화 정보를 Axios 로 가져와서 사용자가 편하게 볼수있게 UI Component를 리엑트를 이용해서 제작하였습니다. 컴포넌트 스타일링은 styled-components를 사용해서 css-in-js 방식으로 스타일링 하였습니다. react-router-dom을 이용해서 TV, Movie, Search, Detail 페이지들을 제작하였습니다 영화를 검색할 수 있거나 클릭해서 영화의 정보를 보여줄 수 있습니다. 호스팅은 Netlify 에 하였습니다.
