이론

<head> 태그는 웹 페이지의 정보를 포현하는 부분입니다. 여기에는 사이트의 여러가지 정보가 들어갑니다. <meta charset="UTF-8">는 웹 페이지의 언어를 표현하는 부분입니다. UTF-8로 설정하여야 모든 언어를 지원합니다. 웹 브라우저의 호환성을 설정하는 부분입니다. 구 익스에 들어갔을 때 브라우저 엔진을 최신 브라우저로 읽어라. 이런 식으로 생각하면 됩니다. 반응형을 고려할 시 PC화면을 모바일 화면에서도 잘 보일 수 있도록 비율을 설정하는 부분입니다.

<title>은 “패럴랙스 스크롤링 사이트 만들기 01-1”로 변경할께요!

<body>는 웹 페이지의 정보를 표현하는 부분입니다.

<main>는 웹 페이지의 핵심 정보 영역입니다. 이 곳에 <section> 태그를 9개 만들고 아이디와 클래스 이름을 설정하겠습니다.
