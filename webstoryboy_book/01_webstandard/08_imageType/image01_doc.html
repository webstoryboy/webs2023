소개

안녕하세요!. 웹스토리보이입니다. 이번 주제는 이미지 유형입니다. 이미지가 주 콘텐츠가 되는 유형입니다. 마우스 오버시 여러가지 효과를 만들 수 있는 구조입니다. 이미지 유형도 사이트 제작시 많이 사용하는 유형이니 잘 만들어 보겠습니다. 그럼 렛츠기릿 🤪

이론

우리가 이번에 만들어 볼 유형은 이런 디자인입니다. 이미지가 두개가 있고, 마우스 오버하면 살짝 효과가 생기는 구조입니다. 지금까지 잘 따라 오셨다면 폴더 구조는 이렇게 되어 있을겁니다. webstandard 폴더에 07_image01.html 페이지를 만들겠습니다.

시멘틱 태그1 <section> 이용하여 배너 영역임을 표현합니다. 클래스 이름은 image-wrap이라고 설정하겠습니다.

가운데 영역을 설정하기 위해서 container 박스를 만들었습니다. 레이아웃 작업할 때 가운데 영역 기억하시죠? 

.image-wrap {
    width: 100%;
    background: rgba(0,0,0,0.2);
}
.container {
    width: 1160px;
    margin: 0 auto;
    padding: 0 20px;
    background: rgba(0,0,0,0.3);
    height: 300px;
}

image-wrap 요소는 전체 영역이기 때문에 width: 100%;을 설정하겠습니다. container는 가운데 정렬2을 위해서 margin: 0 auto; 속성을 설정하겠습니다. 양쪽에 패딩값을 주고 영역을 확인하기 위해서 백그라운드와 높이 값을 가상으로 설정하겠습니다. 

* {
    margin: 0;
    padding: 0;
}

여백을 초기화하기 위해 CSS를 초기화3 하겠습니다.
