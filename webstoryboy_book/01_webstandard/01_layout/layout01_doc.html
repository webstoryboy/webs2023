소개

안녕하세요!. 웹스토리보이입니다. 사이트 만들기 위한 가장 기본적인 레이아웃 유형을 살펴보겠습니다. 레이아웃은 사이트를 만들기 위한 가장 기본적인 단계입니다. 사이트는 레이아웃 속에 레이아웃으로 구조화 되어 있기 때문에 레이아웃을 이해한다면, 사이트 제작이 수월해집니다. 레이아웃을 만드는 방법은 여러가지로 나눌 수 있습니다. table 방식, float 방식, flex 방식, grid 방식으로 나눌 수 있습니다.

table 방식은 2000년 대 이전 방식으로 웹 표준을 준수하지 않은 방법으로 현재는 사용하지 않습니다. float 방식은 2020년 대 이전 방식으로 테이블 방식 대신 이 방법으로 주로 구현되어 왔습니다. 레이아웃을 위한 CSS 속성은 정확하게 존재하지 않았습니다. 그래서 float을 이용하게 되었지만 버그도 발생하고, 레이아웃을 구조화하는데 어려움이 있었습니다. 그래서 현재는 flex 방식과 grid 방식을 사용합니다. 플렉스와 그리드는 레이아웃을 위한 CSS의 새로운 속성이라고 생각하시면 됩니다. 하지만 현재 실무에서는 아직도 float 방식을 많이 사용합니다. 기존에 만들어졌던 사이트들이 float 방식으로 되어 있기 때문에 여러분들은 float 방식도 공부하셔야 합니다. 그래서 저희는 float, flex, grid 세가지 방식으로 레이아웃을 작업할 것입니다. 그럼 준비 되셨죠? 시작해 볼까요! 🤪

이론

이렇게 생긴 디자인을 작업할 것입니다. 레이아웃 유형에서 가장 많이 사용하는 구조로 여러분들도 많이 보셨을겁니다. 레이아웃 방식은 크게 세가지(float 방식, flex 방식, grid 방식)가 있다고 했습니다. 이 방식을 하나씩 작업해보겠습니다. 즉 똑같은 레이아웃을 3가지 방법으로 작업해 볼 것입니다. 그럼 float 방식을 먼저 해보겠습니다.

가장 먼저 VScode에서 webstandard 폴더를 만들고 01_layout01.html 파일을 만들겠습니다.

!를 치고 tab을 누르겠습니다. 언어를 ko로 변경하고 title은 레이아웃 유형01로 변경하겠습니다.

레이아웃 작업을 할 때는 블록 요소1로 레이아웃을 작업합니다. 블록은 하나의 박스라고 생각하면 됩니다. 그림과 같은 레이아웃은 제일 큰 박스 안에 5개의 박스로 구성된 구조로 보시면 됩니다. 결국 레이아웃은 퍼블조각 맞추기라고 생각하시면 쉽니다. 그럼 하나식 따라해보겠습니다.

1. float으로 레이아웃 구성하기

<div>
    <div></div>
    <div></div>
    <div></div>
    <div></div>
    <div></div>

HTML 구조는 이렇게 간단하게 끝납니다. 하지만 똑같은 div 태그로 이루어져 있기 때문에 구별을 해주어야 합니다. 그래서 이름을 지워주겠습니다. 이렇게 이름을 지워주고 선택할 수 있습니다. 이것을 선택자(selector)2라고 합니다. 저는 ID 선택자를 이용하여 이름을 지워주었습니다. 

<div id="wrap">
    <div id="header"></div>
    <div id="nav"></div>
    <div id="aside"></div>
    <div id="section"></div>
    <div id="footer"></div>
</div>

다음은 CSS를 통해서 영역을 잡고, 배경색을 넣어보겠습니다. 

<style>
    #wrap {
        width: 1200px;
    }
    #header {
        width: 1200px;
        height: 100px;
        background-color: #FFE0B2;
    }
    #nav {
        width: 1200px;
        height: 100px;
        background-color: #FFCC80;
    }
    #aside {
        width: 400px;
        height: 780px;
        background-color: #FFB74D;
    }
    #section {
        width: 800px;
        height: 780px;
        background-color: #FFA726;
    }
    #footer {
        width: 1200px;
        height: 100px;
        background-color: #FF9800;
    }
</style>

이렇게 width값과 height 값을 넣어주고, background-color 색을 넣어주면 영역을 확인할 수 있습니다. 한번 확인해 볼까요?

생각했던 것보다는 다르게 나오죠! 😆 우선 div는 블록구조1이기 때문에 한 줄에 하나만 표현이 됩니다. aside 요소와 section 요소는 가로 정렬이 안되어 있기 때문에 가로로 정렬 해주어야 합니다. 그래서 float을 사용 할 것입니다. 

#aside {
    width: 400px;
    height: 780px;
    background-color: #FFB74D;
    float: left;
}
#section {
    width: 800px;
    height: 780px;
    background-color: #FFA726;
    float: left;
}
#footer {
    width: 1200px;
    height: 100px;
    background-color: #FF9800;
    clear: both;
}

#aside 요소와 #section 요소한테 float: left;을 추가하였습니다. #aside 요소와 #section 요소는 정렬이 되었지만 #footer 요소가 없어진 것을 확인 할 수 있습니다. float을 사용하게 되면 버그 한가지가 있습니다. 사실 float의 정확한 용도는 글과 이미지의 정렬 상태를 설정하기 위함이였습니다. 하지만 예전에는 레이아웃을 위한 CSS 속성이 없었기에 임시로 비슷한 float을 사용하게 되었습니다. 그래서 float을 사용하면 높이 값을 인식하지 못합니다. 즉 float을 사용한 박스 영역에 높이 값을 0으로 인식하기 때문에 footer가 사라진 것처럼 보입니다. 그래서 사라진 영역에 clear:both를 사용하면 높이 값이 인식되어 레이아웃이 잡힙니다.

float으로 인하여 영역 깨짐3을 방지하기 위한 방법은 더 있습니다. 그것은 추후 더 알아보면서 공부하도록 하겠습니다.

이제 거의 완성이 되었는데요. 레이아웃이 가운데 와야 합니다. 이것은 블록구조 가운데 정렬4을 이용하면 됩니다. width값이 있는 블록구조는 왼쪽, 오른쪽 바깥쪽 여백을 auto로 설정하면 가운데로 옵니다. 전체 영역을 담당하고 있는 #wrap 요소에게 margin: 0 auto;를 설정하겠습니다. 그리고 마지막으로 body 태그에 background-color: #FFF3E0를 설정하겠습니다.

마지막으로 전체(*) 선택자를 이용하여, 여백을 초기화하겠습니다. 완성된 것을 보면 위쪽에 여백이 있는 것을 볼 수 있습니다. 브라우저마다 기본 태그에는 여백이 존재합니다. 이 값이 브라우저마다 동일하다면 상관이 없지만, 조금씩 다르기 때문에 이 부분을 0으로 초기화 할 것입니다. 보통 이것을 리셋 CSS5 라고 합니다. 이 부분도 설정하는 방법이 다양하지만, 우리는 간단하게만 설정하고 추후 더 자세히 배우도록 하겠습니다. 처음부터 너무 깊이 들어가면 재미없고, 지루하기 때문에 조금씩 조금씩 깊이 들어가면서 반복적으로 배우도록 하겠습니다. 

* {
    margin: 0;
}

3. flex으로 레이아웃 구성하기

기존에 float을 지우고 다시 작업하겠습니다. flex 구조로 작업하기 위해서는 구조 변경도 필요합니다. 물론 위에서 작업할 때도 이렇게 작업하여도 상관없습니다. aside 요소와 section 요소를 설정할 수 있는 상위 박스가 필요합니다. main 요소를 추가하였습니다. 

<div id="wrap">
    <header id="header"></header>
    <nav id="nav"></nav>
    <aside id="aside"></aside>
    <section id="section"></section>
    <footer id="footer"></footer>
</div>

<div id="wrap">
    <header id="header"></header>
    <nav id="nav"></nav>
    <main id="main">
        <aside id="aside"></aside>
        <section id="section"></section>
    </main>
    <footer id="footer"></footer>
</div>

#main를 추가하고 display:flex를 선언하였습니다. 

#main {
    display: flex;
}

flex는 부모 자식관계에서만 영향을 미치기 때문에 aside 요소와 section 요소의 부모 main 요소한테만 flex를 선언하면 끝입니다. float 보다는 간단하죠! 😎

4. grid로 레이아웃 구성하기

이번에는 마지막으로 그리도로 작업하겠습니다. 그리드로 작업하는 방법도 여러가지가 있습니다. 저희는 활용성이 가장 높은 방법으로 작업하겠습니다. 우선 레이아웃은 다시 main 태그가 없는 상태로 만들겠습니다. 

<div id="wrap">
    <header id="header"></header>
    <nav id="nav"></nav>
    <aside id="aside"></aside>
    <section id="section"></section>
    <footer id="footer"></footer>
</div>

gird 역시 부모 자식 관계에서만 영향을 미칩니다. 현재는 부모박스가 #wrap이고 나머지는 자식이 됩니다. 그래서 #wrap에 display: grid;를 선언할 것입니다. 

#wrap {
    width: 1200px;
    margin: 0 auto;
    display: grid;
    grid-template-areas: 
        "header header"
        "nav nav"
        "aside section"
        "footer footer";
    grid-template-columns: 400px 800px;
    grid-template-rows: 100px 100px 780px 100px;
}

grid-template-areas 속성을 이용하여 레이아웃을 구조화 합니다. grid-template-columns 속성을 통해 가로 값을 설정합니다. grid-template-rows 속성을 통해 높이 값을 설정합니다. 이렇게 그리드는 부모 박스에서 영역을 다 설정할 수 있습니다. 그리고 자식 요소한테는 이름만 설정하면 됩니다. 

#header {
    background-color: #FFE0B2;
    grid-area: header;
}
#nav {
    background-color: #FFCC80;
    grid-area: nav;
}
#aside {
    background-color: #FFB74D;
    grid-area: aside;
}
#section {
    background-color: #FFA726;
    grid-area: section;
}
#footer {
    background-color: #FF9800;
    grid-area: footer;
}

각 각의 요소에 이름만 설정해주면 그리드 레이아웃은 끝이 납니다. 어떤가요? 어렵지 않죠? 😚 마지막 결과랑 같은지 확인해 보세요~
