소개

안녕하세요!. 웹스토리보이입니다. 우리가 최종적으로 만들 사이트입니다. 현재 우리나라에서 만들어진 사이트는 종류별로 살펴본다면, 크게 정적인 사이트, 동적인 사이트로 나눌 수 있을 것 같습니다.

세부적으로 더 나눌수도 있지만 크게 본다면 두가지로 구별됩니다. 정적인 사이트라 함은 필요한 정보만 딱 보여주는 정부기관 사이트가 대표적일 것 같습니다. 동적인 사이트라 함은 필요한 정보를 여러가지 효과 통해서 보여주는 사이트를 말합니다. 가장 대표적으로 웹 에이전시에서 만드는 패럴랙스형 사이트입니다. 스크롤을 내리면 효과가 나타나는 사이트를 말합니다.

우리가 만들 "웹 표준 사이트 만들기"는 정적인 사이트에 가깝습니다. 정적인 사이트든, 동저인 사이트든 중요한건 콘텐츠 내용입니다. 콘텐츠 내용을 어떻게 보여주는 것이 핵심이라고 할 수 있습니다. 정적인 사이트를 바탕으로 스크립트를 통해 움직임을 주면 동적인 사이트가 되는 것입니다.

그래서 우리는 콘텐츠 내용을 유형별로 코딩을 할 것입니다. 유형별로 완성한 코드를 하나로 합치면 우리가 만들 "웹 표준 사이트"가 되는 것입니다. 유형에는 메뉴 유형, 배너 유형, 카드 유형, 이미지 유형, 이미지/텍스트 유형, 푸터 유형 등으로 나눌 수 있으며, 여기에 스크립트를 추가하면 슬라이드 유형, 탭 유형이 됩니다. 그럼 이제부터 하나씩 하나씩 배워보도록 하겠습니다.

<meta name="robots" content="all">

head 요소 안에는 사이트에 대한 정보를 제공합니다. SEO(검색엔진최적화)1 사이트에서 가장 중요할 지 모릅니다. 정성스럽게 사이트를 만들었는데 아무도 들어와 주지 않으면, 슬프겠죠! 그래서 검색에 잘 노출 될 수 있도록 meta 정보을 head 요소에 제공해야 합니다. 

<!-- CSS(스타일) -->
<link rel="stylesheet" href="assets/css/fonts.css">
<link rel="stylesheet" href="assets/css/reset.css">
<link rel="stylesheet" href="assets/css/common.css">
<link rel="stylesheet" href="assets/css/style.css">

다음은 CSS를 연동할 것입니다. 지금까지는 CSS를 head 요소 안에 style을 선언하여 넣어지만 이제는 따로 파일을 만들어서 연동할 것입니다. 파일이 많아지기 때문에 외부 파일로 만들어서 작업할 것입니다. 이렇게 CSS 선언하는 방법2은 총 3가지 있습니다. 그 중에서 2번째 방법까지 완료한 것입니다. 나머지는 추후에 또 나올 것입니다.

fonts.css는 웹 폰트 관련된 CSS를 정리할 것이고, reset.css는 초기화와 관련된 CSS를 정리할 것이고, common.css는 공통적으로 사용되는 CSS를 정리할 것이고, sytle.css는 유형별로 코딩한 CSS를 정리할 것입니다. 사이트 규모가 커지면 더 세분활 될 수도 있습니다. 이런 부분이 본인이 하기 나름입니다. 정답이 있는 것은 아닙니다. 코딩을 많이 하다보면 자기만의 노하우가 쌓일 것입니다. 회사에 들어갔을 때는 회사 규율에 맞게 작업하시면 됩니다.

영역의 헤더 영역, 메인 영역, 푸터 영역으로 나누고, 메인 영역은 다시 슬라이드 유형, 이미지 유형, 카드 유형, 이미지텍스트 유형, 배너 유형, 텍스트 유형으로 나눌 것입니다. 헤더 영역에는 메뉴 유형이 들어가고, 푸터 영역에는 푸터 유형이 들어갈 것입니다. 각각의 영역은 section 태그로 설정하였고, 이름도 아이디로 설정하였습니다.

그럼 우리는 이제 유형별로 하나씩 작업하고 이 사이트에 접목을 시킬 것입니다. 순서는 제일 쉬운 배너 유형부터 배고 오겠습니다.

어떤가요? 저랑 비슷하게 나오나요? 그럼 나머지 여백 설정과 폰튼 설정도 하겠습니다. 

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* 여백 초기화 */
body, div, dl, dt, dd, ul, ol, li, h1, h2, h3, h4, h5, h6, pre, code, form, fieldset, legend,
textarea, p, blockquote, th, td, input, select, button, figure, figcaption {
    margin: 0;
    padding: 0;
}

/* 박스 사이즈 초기화 */
*,
* : before,
* : after {
	box-sizing: border-box;
}

reset.css 여백 초기화 소스를 오른쪽 처럼 넣어주겠습니다. 간단하게 작업할 때에는 왼쪽처럼 해주었지만, 실제 사이트 제작시에는 해당 태그를 직접 선택하여, 초기화를 하였습니다. 박스 사이즈 초기화3 작업은 모든 요소한테 설정해 줄 것이며, CSS 가상요소4를 통해서도 설정할 것입니다. 모든 요소 가로 값에는 padding 값이 포함 되어 있는데, 이 부분에 패딩값을 제외하여 설정하는 것이 박스 사이즈 초기화라고 볼 수 있습니다.
